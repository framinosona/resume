# GoneToLyon - DevKings National Winner

**GoneToLyon** won the **1st National Prize in all categories** at Microsoft's DevKings Challenge. This innovative tourism app used augmented reality and open data to transform how visitors explore Lyon, France.

## Project Innovation

### Vision

Create an immersive tourism experience that combines real-world navigation with digital information overlay, making city exploration more engaging and informative.

### Core Features

- **Augmented Reality Navigation**: Real-time AR overlays on camera view
- **Open Data Integration**: Live information from Lyon's public data sources
- **Interactive Points of Interest**: Detailed information about landmarks and attractions
- **Personalized Tours**: Customized routes based on user preferences
- **Social Sharing**: Share discoveries and experiences with friends

## Technical Achievement

### Augmented Reality Implementation

- **Computer Vision**: Real-time object recognition and tracking
- **Spatial Mapping**: 3D world understanding for accurate overlay placement
- **Performance Optimization**: Smooth AR experience on mobile devices
- **Cross-Platform Compatibility**: Consistent experience across Windows devices

### Open Data Utilization

- **Real-Time APIs**: Integration with Lyon's open data platform
- **Data Visualization**: Creative presentation of municipal information
- **Location Services**: GPS-based contextual information delivery
- **Offline Capabilities**: Cached data for areas with poor connectivity

## Competition Success

### Microsoft Recognition

This project demonstrated cutting-edge use of Microsoft technologies and won recognition across all competition categories:

- **Technical Innovation**: Advanced AR implementation
- **Design Excellence**: Intuitive user interface and experience
- **Business Potential**: Clear commercial viability
- **Social Impact**: Enhanced tourism and cultural engagement

### Industry Impact

GoneToLyon showcased the potential of augmented reality in tourism before AR became mainstream, influencing future development in location-based AR applications.

## Technology Stack

### Core Platforms

- **Universal Windows Platform (UWP)**: Modern Windows app development
- **Windows 8.1**: Target platform optimization
- **XAML**: Rich user interface development
- **C#**: Application logic and AR algorithms

### AR and Data Technologies

- **Computer Vision APIs**: Object recognition and tracking
- **Location Services**: GPS and mapping integration
- **Open Data APIs**: Lyon municipality data sources
- **Cloud Services**: Backend data processing and storage

This project established early expertise in augmented reality development and demonstrated the power of combining emerging technologies with public data to create meaningful user experiences.
