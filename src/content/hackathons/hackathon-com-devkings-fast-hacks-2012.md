# DevKings Fast Hacks 2012 - 8tris Design Winner

**8tris** was our award-winning Windows 8 UI design that earned **1st Design Prize** at DevKings Fast Hacks 2012, Microsoft's fast-paced design challenge focusing on the revolutionary new Metro design language and touch-first interface paradigms.

## Project Overview

### Design Challenge Context

Windows 8 represented a fundamental shift in Microsoft's design philosophy:

- **Metro Design Language**: Clean, typography-focused, grid-based layouts
- **Touch-First Interface**: Optimized for tablet and touch interactions
- **Live Tiles**: Dynamic, information-rich application tiles
- **Immersive Experience**: Full-screen applications with minimal chrome

### 8tris Concept

Our design reimagined the classic Tetris concept through the Windows 8 design lens:

- **Modern Interpretation**: Classic gameplay with contemporary visual design
- **Touch Optimization**: Gesture-based controls and touch-friendly interactions
- **Live Tile Integration**: Dynamic game state representation on Start screen
- **Metro Aesthetics**: Bold typography, flat design, and vibrant color schemes

## Design Innovation

### Metro Design Language Implementation

#### Visual Design Principles

- **Typography First**: Used typography as primary design element
- **Grid-Based Layout**: Precise alignment following Metro grid system
- **Color and Contrast**: High-contrast color schemes for accessibility
- **Minimal Chrome**: Reduced interface clutter for content focus

#### Touch Interface Design

- **Gesture Integration**: Swipe, tap, and pinch gestures for game control
- **Thumb-Friendly**: Controls positioned for natural thumb reach
- **Visual Feedback**: Clear response to touch interactions
- **Responsive Layout**: Adaptive design for different screen orientations

### Game Experience Innovation

#### Modern Tetris Interpretation

- **Simplified Controls**: Streamlined game mechanics for touch devices
- **Visual Enhancement**: Particle effects and smooth animations
- **Progressive Difficulty**: Adaptive challenge scaling
- **Social Integration**: Sharing capabilities and achievement system

#### Windows 8 Integration Features

- **Live Tiles**: Real-time game statistics and high scores
- **Snap View**: Gameplay in snapped application state
- **Search Contract**: Game-specific search functionality
- **Settings Charm**: Integrated Windows 8 settings experience
