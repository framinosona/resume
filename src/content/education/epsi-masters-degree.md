# IT Engineering Program - EPSI Lyon

Completed comprehensive 5-year engineering program at EPSI Lyon, progressing from technical foundations through advanced specialization in Mobile-IoT-Robotics. This intensive program combined theoretical computer science education with practical industry partnerships and competitive programming.

## Program Structure

### Years 1-2: Technical Foundations (2011-2013)

- **BTS Services Informatiques aux Organisations**: State diploma providing foundational IT skills
- **Core Curriculum**: Programming fundamentals, database management, network administration
- **Professional Skills**: IT project management, system analysis, technical communication

### Year 3: Computer Science Bachelor (2013-2014)

- **Bachelor en Informatique**: Advanced computer science concepts and software engineering
- **Specialization Preparation**: Bridge to engineering-level studies
- **Industry Readiness**: Advanced programming, algorithms, software architecture

### Years 4-5: Master's Specialization (2014-2016)

- **Mobile-IoT-Robotics Major**: Cutting-edge specialization in emerging technologies
- **Industry Partnerships**: Microsoft Student Partner and Xamarin Student Partner programs
- **Research & Innovation**: Advanced projects in mobile development and IoT systems
